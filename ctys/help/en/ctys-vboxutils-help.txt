
NAME
====

ctys-vboxutils - VirtualBox utilities

SYNTAX
======

ctys-vboxutils


   [-d <debug-options>]
   [-h]
   [-H <help-options>]
   [-n]
   [-V]
   [-X]
   [
     (
       (FETCH|F) (
           VBOXPATH4ID|P4I
           |VBOXUUID4PATH|UUID
           |VBOXNAME|NAME
           |VBOXCFGFILE|CFGFILE
           |VMSTATE|STATE|VS
           |VMMAC|MAC
           |VMRDP|VR
           |VMRDPLST|VRL
        )
     )
     |
     (
       VMMANAGE (
           GETCFGFILE
        )
     )
     |
     (
       (LIST|L) (
           SERVERS_RAW
           |CLIENTS_RAW
       )
     )
   ]

DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
