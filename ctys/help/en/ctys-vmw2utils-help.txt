
NAME
====

ctys-vmw2utils - VMW-Server2 utilities

SYNTAX
======

ctys-vmw2utils


   [-d <debug-options>]
   [-h]
   [-H <help-options>]
   [-n]
   [-V]
   [-X]
   [
     (
       (FETCH|F) (
           VMWPATH4OBJID|P4O
          |VMWOBJID4PATH|O4P
          |DATASTORE|D
        )
     )
     |
     (
       (CONVERT|C) (TODATASTORE|2D)
     )
     |
     (
       (LIST|L) (
           INVENTORY|I
          |DATASTORES|D
       )
     )
   ]

DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
