
NAME
====

ctys - Commutate To Your Session

SYNTAX
======

ctys


   [<action>]
   [<generic options>]
   [--]
   <arguments>


DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
