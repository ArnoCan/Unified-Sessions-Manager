
NAME
====

ctys-extractMAClst -  generates a list of mappings DHCP

SYNTAX
======

ctys-extractMAClst


   [-d <debug-args>]
   [-E]
   [-n|-i|-m]
   [-h]
   [-H <help-options>]
   [-p <db-dir-path>]
   [-P]
   [-q]
   [-V]
   [-X]
   <pathname-of-dhcpd.conf>


DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
