
NAME
====

ctys-install - base interface for installation and update

SYNTAX
======

ctys-install


   [libdir=<default=\$HOME/lib>]
   [bindir=<default=\$HOME/bin>]
   [templatedir=<default=\$HOME/ctys>]
   [remove<default=unset>]
   [noconf<default=unset>]
   [force<default=unset>]
   [forceall<default=unset>]
   [forceclean<default=unset>]
   [linkonly<default=unset>]
   [version|--version|-V]

   [-h]
   [-H <help-options>]
   [-X]


DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
