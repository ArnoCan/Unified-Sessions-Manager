
NAME
====

ctys-vdbgen - generates databases of VMs and PMs

SYNTAX
======

ctys-vdbgen


   [--append]
   [--base=<base-path-list>]
   [--cacheDB=<output-db-directory-path>]
   [--filecontext]
   [--progress]
   [--progressall]
   [--replace]
   [--stdio]

   [-b]
   [-c]
   [-C]
   [-d <debug-level>]
   [-h]
   [-H <help-options>]
   [-t]
   [-T]
   [-V]
   [-X]
   <target-list>



DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
