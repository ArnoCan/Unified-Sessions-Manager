
NAME
====

ctys-groups -  lists groups from groups DB and groups caches

SYNTAX
======

ctys-groups


   [-c]
   [-d <level>]
   [-D <deepness-level>]
   [-F]
   [-h]
   [-H <help-options>]
   [-l <remote-user>]
   [-L <remote-user>]
   [-m <1|2|3|4|5>]
   [-R <remote-hosts>]
   [-S]
   [-V]
   [-X]
   [<group-list>]




DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
