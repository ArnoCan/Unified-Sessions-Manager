
NAME
====

ctys-groups -  lists groups from groups DB and groups caches

SYNTAX
======

ctys-groups


   [-c]
   [-d <level>]
   [-D <deepness-level>]
   [-e [<group-list>|<path-list>]]
   [-F]
   [-h]
   [-H <help-options>]
   [-l]
   [-m <1|2|3|4|5|6|7|7u|8|8u>]
   [-S]
   [-V]
   [-X]
   [<group-list>]




DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
