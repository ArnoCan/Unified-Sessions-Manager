
NAME
====

ctys-smbutil - displays available SMB server and workstations

SYNTAX
======

ctys-smbutil


   [-d <debug-level>]
   [-h]
   [-H <help-options>]
   [-i <numerical-IP>]
   [-l <remote-login>]
   [-n <dns-TCP>]
   [-r [(REVERSE|R|-),]PING|SSH[,PM|VM] ]
   [-V]
   [-X]

DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
