
NAME
====

ctys-config -  lists and edits configuration files

SYNTAX
======

ctys-config


   [-d <level>]
   [-D <deepness-level>]
   [-e [<group-list>|<path-list>]]
   [-F]
   [-h]
   [-H <help-options>]
   [-S]
   [-V]
   [-X]
   [<configfile-list>]



DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
