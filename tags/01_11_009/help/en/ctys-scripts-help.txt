
NAME
====

ctys-scripts -  lists scripts

SYNTAX
======

ctys-scripts


   [-d <level>]
   [-e [<scripts-list>|<path-list>]]
   [-h]
   [-H <help-options>]
   [-l]
   [-m [<0>]]
   [-V]
   [-X]
   [<scripts-list>]




DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
