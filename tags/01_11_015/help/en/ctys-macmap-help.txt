
NAME
====

ctys-macmap - searches within the given list of "macmap.fdb"

SYNTAX
======

ctys-macmap


   [-d <debug-options>]
   [-n|-i|-m]
   [-h]
   [-H <help-options>]
   [-p <db-dir-path-list>]
   [-u]
   [-V]
   [-X]
   <simple awk-regexpr>


DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
