
NAME
====

ctys-genmconf -  generates the configuration entry

SYNTAX
======

ctys-genmconf


   [--(brdglist|--bl)=<contained-if-prefix>]
   [--(brdglistxen|blx)=<contained-if-prefix>]
   [--(brdgcheckxen|bcx)=<bridge>]
   [-d <debug-options>]
   [-h]
   [-H <help-options>]
   [--iflist|--il}
   [--iffirst|--if]
   [--(iflistnames|iln)[=(ALL|PREFIX|POSTFIX|MATCH)[,<namestring>]]]
   [--iflistmacip|--ilmi]
   [-L <remote-user>]
   [-R <remote-host>]
   [-V]
   [-X]




DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
