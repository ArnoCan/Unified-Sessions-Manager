
NAME
====

ctys-cloneVM - Copy and configure VMs

SYNTAX
======

ctys-cloneVM


   [-d <debug-level>]
   --directory=<directory>
   [--defaults-file=<file-name>]
   [-f]
   [-h]
   [-H <help-options>]
   [--ip=<IP-address>]
   --label=<the-target-vm-label>
   --label-old=<the-source-vm-label>
   [--mac=<MAC-address>]
   [--no-create-image]
   [--no-save-para-kernel]
   -t <session-type>
   [--tcp=<TCP-DNS-name>]
   [-V]
   [--vm-state=<VSTAT>]
   [--uuid=<UUID>]
   [-X]



DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
