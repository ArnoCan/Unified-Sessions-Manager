
NAME
====

ctys-xdg - Manages FreeDesktop entries

SYNTAX
======

ctys-xdg


   [-d]
   [-e]
   [--force]
   [--helpEx|-helpEx]
   [-h|--help|-help]
   [--menu-cancel]
   [--menu-cancel=<config-template-root-or-home-pathname>]
   [--menu-create]
   [--menu-create=<config-template-root-or-home-pathname>]
   [--menu-edit]
   [--menu-edit=<files-or-directories>]
   [--menu-gui]
   [--menu-private]
   [--menu-private=<target-base>]
   [--menu-shared]
   [--menu-shared=<target-base>]
   [--no-user]
   [--user]
   [-V]
   [-X]

DESCRIPTION
===========

Refer to documentation or use following option for
extended help:

     ctys -H help

     displays available online help options.
